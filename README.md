# 주식 투자 분석 대시보드

React + TypeScript + Tailwind CSS로 구현한 주식 투자 분석 대시보드

## 주요 기능

- 📊 코스피 200 종목 실시간 조회
- 📈 투자지표 분석 (PER, PBR, ROE) + 업종 평균 비교
- 🤖 AI 기반 투자 분석 페이지
- ⭐ 관심종목 관리 (로그인 필요)
- 🔍 종목 검색 기능
- 📰 종목별 뉴스 패널
- 💹 업종별 시장 동향

## 설치 및 실행

```bash
# 패키지 설치
pnpm install

# 개발 서버 실행
pnpm dev

# 빌드
pnpm build
```

## 기술 스택

- **Frontend**: React 18, TypeScript
- **Styling**: Tailwind CSS v4
- **Charts**: Recharts
- **Routing**: React Router v7
- **Build Tool**: Vite
- **Package Manager**: pnpm

## 프로젝트 구조

```
src/
├── app/
│   ├── components/     # 재사용 컴포넌트
│   ├── data/          # 종목 데이터
│   ├── pages/         # 페이지 컴포넌트
│   └── App.tsx        # 메인 앱
├── styles/            # 스타일 파일
└── imports/           # 이미지 에셋
```
